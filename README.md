# Zepy-Chat App

## This app is a fullstack app that serves many standard features

This app has a wide range of modern features, including:
- Dark Mode
- Authentication
- Filtering
- Querying
- I18Next
- And much more...

## Technology Stack

This app is built using the following technologies:
- Frontend:
  - React
  - React-Query
  - Tailwind CSS
  - Redux
- Backend:
  - Express.js
  - Socket.IO
